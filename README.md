🚀 Overview

This project presents an interactive Power BI dashboard built to analyze banking customer data and uncover key insights related to customer demographics, income distribution, and financial behavior. The analysis combines MySQL, Python (Jupyter Notebook), and Power BI to transform raw data into actionable insights.

🛠️ Tools & Technologies
🐍 Python (Pandas, NumPy) – Data cleaning & preprocessing
🗄️ MySQL – Data storage & querying
📓 Jupyter Notebook – Data analysis (EDA)
📊 Power BI – Dashboard creation & visualization


📂 Dataset Features

The dataset includes attributes such as:

Customer ID
Age
Gender
Income / Estimated Salary
Occupation
Nationality
Credit-related attributes
